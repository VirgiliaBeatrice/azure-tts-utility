# Project Title

azure-tts-utility

## Description

A console utility for Azure Text-To-Speech service.

## Getting Started

### Dependencies

Windows concole application

### Installing

Download from [Releases]()

### Executing program

```Powershell
azure-tts-utility -i "<your-identity-file>" -a "<target-audio-file>" -t "<content-file>"
```

## Help

```
azure-tts-utility -h
```
